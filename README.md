# MealPrepAI

## Starting Frontend and Backend Locally

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
    npm install
   ```
3. Start the backend server:
   ```bash
   node dist/index.js
   ```

### Frontend

1. Navigate to the `MealPrepAI_App` directory:
   ```bash
   cd MealPrepAI_App
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Expo development server:
   ```bash
   npm start
   ```
